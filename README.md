# Demo3101
"Управление персоналом"

## Вступление
Выполнение варианта демонстрационного экзамена во время учебной практики.

## Задача
Создать desktop-приложение для управления персоналом некой компании. Приложение состоит из четырёх окон: Навигационное меню, Окно организационной структуры компании и списка с персоналом, Окно информационного раздела компании, Окно добавления/редактирования информации о персонале.

## Работа с репозиторием
В проекте две ветки: master2 (основная) и master. Используйте ветку master2. В этой ветке есть три папки - в одной из них проект, а в остальных условия задания.

Папки и файлы в ветке:
- Demo0203 (папка с проектом);
- Task_02_26/Сессия 4 (Часть задания: Сессия 4);
- task01.31 (Часть задания: Сессия 1 и Сессия 2);
- calendar.png (изображение для интерфейса);
- calendar.txt (файл с кодом для реализации рабочего календаря);
- save.png (изображение для интерфейса).
